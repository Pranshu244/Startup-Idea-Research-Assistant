# Startup-Idea-Research-Assistant
A multilingual AI‑powered chatbot that helps you analyze and refine your startup ideas. Simply upload your idea as a PDF document, and the assistant will break it down, extract insights, and provide conversational feedback to guide your next steps.
## How to Use the Startup Idea Research Assistant??

1. Open the folder **`Chatbot_Code`**.  
2. Click on **`Startup-Idea-Research-Assistant.ipynb`**.  
3. Select **“Open in Colab”**.  
4. Once the Colab page opens, connect your runtime.  
5. Upload your startup idea as a **PDF file**.  
6. Insert your Gemini API key in **Cell 6** at the designated location.  
   - You can generate an API key at [Google AI Studio](https://aistudio.google.com).  
7. Enter your PDF file name in **Cell 7** (e.g., `AI_Startup_Document.pdf`).  
8. Click on **“Run all”** to execute the notebook and click **"Run anyway"**.  
9. Wait a few minutes for all cells to finish running.  
10. If you see a warning like this, press **Cancel**:  
    ```
    Restart session
    WARNING: The following packages were previously imported in this runtime: [packaging]
    You must restart the runtime in order to use newly installed versions.
    Restarting will lose all runtime state, including local variables.
    ```
11. After execution, you will see a **URL** in the output of the last cell.  
12. Click the URL to open the chatbot web app.  
13. Start chatting with the assistant about your startup idea (from the PDF you uploaded).
